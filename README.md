# 🕹️ Hangman Game

A simple and interactive **Hangman game** built using **HTML, CSS, and JavaScript**, where players guess a hidden word letter by letter within a limited number of attempts.

---

## 🚀 Features
- Random word selection for each game  
- Letter-by-letter guessing system  
- Limited attempts with visual feedback  
- Displays correct and incorrect guesses  
- Restart / New Game functionality  
- Responsive design for desktop and mobile  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## 📌 How to Play
1. Start the game and view the hidden word as underscores.  
2. Guess letters one at a time.  
3. Correct guesses reveal letters in the word.  
4. Incorrect guesses reduce remaining attempts.  
5. Win by guessing the word before attempts run out.
